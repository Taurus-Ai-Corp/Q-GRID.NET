## Canadian Case Law Compliance

**Amazon.com v. Canada (2011 FCA 328)**
- All patents emphasize "technological improvement" over business methods
- Technical implementations (ML-DSA, Groth16, multi-agent coordination) highlighted

**Schlumberger v. Commissioner (1982 SCC)**
- Claims structured as "computer-readable medium storing instructions"
- Include processor, memory, network interface components
- Avoid pure algorithm claims

**CIPO PN 2020-04 (Computer-Implemented Inventions)**
- Each patent solves "technological problem" using "technological means"
- Clear technical problem statements in Background sections
